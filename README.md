# üöÄ `Quick Start`

### üíø Installation

```bash
npm install web3uikit
```

or

```bash
yarn add web3uikit
```

### üß∞ Usage

```jsx
import { CryptoCards, Button } from 'web3uikit';

const App = () => (
    <>
        <CryptoCards chain="ethereum" />
        <Button theme="primary" type="button">
            Launch dApp
        </Button>
    </>
);
```
